Student
API

Minimal student management interface designed for handling student records through low-level socket-based communication. The system focuses on speed, simplicity, and direct API interaction without unnecessary layers or frontend frameworks.

Project Description
Student API is a lightweight backend project built using pure C socket programming for handling HTTP requests and managing student records through REST methods.

The frontend is developed only with HTML and CSS, following a hyperminimal black-and-white aesthetic focused on speed, clarity, and low overhead.
Pure socket architecture • C backend • HTML/CSS frontend • REST communication • Minimal footprint

Store student records directly through the socket API server.

Retrieve Student

About The Project
This project demonstrates HTTP communication using low-level socket programming in C without external frameworks. The API manually handles request parsing, routing, and response generation through raw TCP connections.

The frontend remains intentionally minimal to keep the focus on backend communication and REST structure.
